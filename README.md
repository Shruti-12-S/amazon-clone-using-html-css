# 🛒 Amazon Clone (HTML & CSS)

A responsive front-end clone of the **Amazon homepage** built using pure **HTML5** and **CSS3**.
This project replicates the layout, structure, and styling of Amazon’s landing page to practice modern web design concepts like Flexbox, positioning, and responsive layouts.

---

## 📌 Project Overview

This Amazon Clone includes:

* Navigation bar with logo, address, search bar, sign-in, returns, and cart
* Secondary panel menu (Deals, Customer Service, etc.)
* Hero section with promotional banner
* Shopping category grid section
* Multi-column footer similar to Amazon
* Hover effects and styled UI components

The project focuses purely on **frontend UI design**, without JavaScript functionality.

---

## 🛠️ Built With

* **HTML5** – Structure of the webpage
* **CSS3** – Styling and layout
* **Font Awesome** – Icons (cart, location, search, menu)

---

## 🎯 Features

✔️ Amazon-style navigation bar

✔️ Flexbox layout for responsive sections
